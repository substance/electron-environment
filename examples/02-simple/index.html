Example: simple
This is a pretty simple example. Still it needs some techniques to achieve just these basics:
  • SUBSTANCE
  • window.onload: used to do things with Javascript after the page has been loaded
  • document.querySelector(): used to select an element present in the DOM. Used with the same syntax as CSS rules are applied. See Document.querySelector() and Selectors for details.
  • el.addEventListener(): used to register to DOM events, such as a click. There is a bunch of events available: Web/Events In general https://developer.mozilla.org is the best reference for HTML and Javascript.
  • postion: relative and postion: absolute: used to render an element out of the flow. The nearest element with postion: relative defines the rectangle in which the element is positioned using a combination of top, bottom, left, and right
  • FontAwesome icons: a nice set of icons available for free that can be used for buttons and such. You need to include a font-awesome CSS and use it such as in <i class="fa fa-home"></i>. Find all available icons here
